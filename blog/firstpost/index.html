I’ve finally done something with my website

I’ve finally gotten around to doing something productive-ish, with my website/domain that I’ve had for the past few years!

Here’s the tech bits anyone might care about:

  • Built with Hugo using the default ananke theme as a base, and customizing it for what I wanted.
    • One thing I wanted was to be able to write simple content articles in something easy to understand a preview, and Hugo lets me write markdown articles, have a base theme where overwriting and catering it to what you want is incredibly easy and mostly intuitive.
  • Hosted on Github Pages, which I can’t recommend enough for devloper/account/project level hosting of static content
  • Comment system using Utterences
    • tl;dr; Uses github issues to track comments based on the page. I chose this because I didn’t want to deal with privacy issues, pay for hosting a backend service, or deal with having some service own anyone’s comments. This way I have complete control over what happens with any community involvement.

Beyond this, I want to make this the centerpoint for tutorials, thoughts, and ideas that I want to put forth in the world. If I can help anyone with development or design along the way, then that’s even better.
